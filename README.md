# Learnova Academy 🎓

Learnova Academy is a responsive online learning platform inspired by Udemy, developed using HTML and CSS. This project was created to practice front-end web development concepts such as Flexbox, responsive layouts, and modern UI design.

## 🚀 Features

- Responsive Navigation Bar
- Search Functionality
- Course Categories Section
- Recommended Courses Section
- Popular Courses Section
- Topics Recommendation Section
- Promotional Banner
- Footer with Quick Links
- Clean and User-Friendly Interface

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome
- Google Fonts (Gabarito)

## 🎯 Learning Outcomes

Through this project, I learned:

- HTML page structure
- CSS styling and layouts
- Flexbox positioning
- Responsive web design basics
- Creating a modern educational website UI
